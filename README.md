# SistemaNotasporUsuario
Proyecto de la actividad evaluativa de la UT5

El objetivo es crear un sistema que permita registrar usuarios, iniciar sesión y gestionar notas usando ficheros

# Contenidoproyecto
El proyecto incluye:
Dos clases de modelo;
    Usuario
    Nota
Dos controllers;
    AuthController
    NotaController
Tres clases de utils;
    FileUtils
    InputUtils
    SecurityUtils
Un Main; que ejecuta todo el proyecto
